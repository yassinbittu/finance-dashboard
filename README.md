# Finance Dashboard

A frontend Finance Dashboard built using React and Tailwind CSS.

## Features
- Summary Cards (Balance, Income, Expenses)
- Charts (Trend & Category Breakdown)
- Transactions Table with Search, Filter, Sort
- Role Based UI (Viewer & Admin)
- Admin Add Transaction Modal
- Insights Section
- State Management using Context API
- Responsive Design

## Tech Stack
- React.js
- Tailwind CSS
- Recharts
- Context API

## Run Project
npm install
npm run dev
